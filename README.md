<h1 align="center">
  Deck of cards
</h1>

1.  **Clone Repo**

    ```shell
    git clone https://github.com/muscaiu/deck-of-cards
    ```

2.  **Install dependecinies**

    ```shell
    npm install
    ```

3.  **Start**

    Start in dev mode: http://localhost:8000

    ```shell
    npm run develop
    ```

    Or start in prod mode:http://localhost:9000

    ```shell
    npm start
    ```
